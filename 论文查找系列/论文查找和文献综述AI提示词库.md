# 论文查找和文献综述 AI提示词库

## 📚 使用说明

这是一个专门用于帮助你**快速找到相关领域的顶级论文**的Prompt库。

**适用场景：**
- 你已经确定了选题
- 需要快速找到近几年的相关论文
- 想了解哪些是该领域的Top期刊和会议
- 需要给文献综述提供方向

**工具组合：**
- ChatGPT/Claude：用这些Prompt找论文方向和期刊信息
- Google Scholar：根据推荐搜索实际的论文
- Web of Science/Scopus：验证期刊等级

---

## 一、确定Top期刊和会议

### Prompt 1.1 - 找出你研究领域的顶级期刊（强烈推荐）

```
我的研究方向是：[你的研究方向/选题]

我需要你帮我找出这个领域最权威的期刊和会议。

请给我：

【第一部分：顶级期刊（International Level）】
列出5-8个该领域的顶级国际期刊，包括：
- 期刊名称（中英文）
- 影响因子（IF）或类似指标
- 主要关注的研究方向
- 出刊周期
- 审稿周期
- 该期刊在领域中的地位和特点

【第二部分：优质期刊（High Quality）】
列出5-8个该领域的优质但可能稍次于顶级的期刊
- 同样的信息格式

【第三部分：顶级会议（A类会议）】
列出3-5个该领域最重要的国际会议，包括：
- 会议名称（中英文 + 缩写）
- 举办地点和周期（年度/两年一次等）
- 接收论文方向
- 影响力（H-index或引用情况）
- 录取率
- 最新一次会议的时间

【第四部分：高质量会议（B类会议）】
列出3-5个优质会议

【第五部分：选择建议】
根据我的研究方向，推荐我应该重点关注的3-5个期刊/会议，并说明理由
```

**使用建议：**
- 这个Prompt会给你完整的期刊和会议清单
- 优先看"顶级期刊"和"顶级会议"的推荐
- 记下这些期刊和会议的名字，后续在Google Scholar中搜索

### Prompt 1.2 - 了解期刊分级体系

```
在[学科领域]中，有什么期刊评级系统吗？

请解释：

   - 影响因子（IF）？
   - SJR？
   - H-index？
   - 其他指标？

2. 各个等级通常对应什么样的IF或指标值？
   - Top期刊通常IF是多少？

   - 一般期刊的IF范围是多少？

3. 除了IF还有什么其他考虑因素？
   - 是否需要看是否被SSCI/SCI收录？
   - 是否需要看被引用次数？
   - 国内评价系统中的分级？

4. 对于硕士论文，应该优先投哪一级的期刊？

5. 推荐我查看这些资源来找期刊信息：
   [给出具体的网站/数据库]
```

## 二、快速查找相关论文

### Prompt 2.1 - 生成论文搜索策略

```
我的研究选题是：[你的选题名称]

我需要在Google Scholar、PubMed、IEEE等数据库中搜索相关论文。

请帮我制定一个搜索策略，包括：

【第一部分：核心关键词】
列出10-15个最相关的核心关键词/短语（英文）：
- 3-5个最核心的关键词
- 5-7个相关但稍次要的关键词
- 2-3个可能相关的拓展关键词

并标注：
- 每个关键词为什么重要
- 这个关键词的中文翻译
- 常见的同义词或相关词

【第二部分：搜索式组合】
给我5-8个具体的搜索式，可以直接复制到Google Scholar：

例如：
- 搜索式1："[关键词1]" AND "[关键词2]"
- 搜索式2：[关键词3] OR [关键词4]
- 搜索式3：[关键词5] NOT [排除词]

对每个搜索式标注预计会找到什么类型的论文

【第三部分：搜索渠道】
推荐我在哪些数据库搜索：
□ Google Scholar
□ Web of Science
□ Scopus
□ 学科专业数据库（如[学科]）
□ 其他

【第四部分：时间范围建议】
- 应该查最近几年的论文？
- 是否需要查经典论文（5-10年前）？
- 为什么这样选择？

【第五部分：筛选标准】
找到论文后，用什么标准判断是否值得读？
- 期刊/会议等级
- 引用次数
- 作者知名度
- 其他
```

**使用建议：**
- 复制这个Prompt的关键词到Google Scholar搜索框
- 这会帮你节省很多时间

### Prompt 2.2 - 快速筛选相关论文

```
我找到了一些论文，需要快速判断哪些值得深入阅读。

请给我一个"论文筛选清单"，我可以用来快速评估论文的价值：

【筛选标准1：来源质量】
☐ 期刊/会议是否是Top期刊/会议？
☐ 期刊的IF或排名如何？
☐ 作者是否来自知名研究机构？
☐ 作者是否是该领域的知名学者？

【筛选标准2：相关性】
☐ 题目和摘要是否直接相关我的研究？
☐ 是否解决了我关心的问题？
☐ 是否用了我想要的研究方法？
☐ 研究对象是否和我的研究对象相同或相近？

【筛选标准3：影响力】
☐ 被引用的次数多吗（同类型论文的平均水平）？
☐ 是否被该领域的其他重要论文引用？
☐ 发表时间是最近的吗？

【筛选标准4：深度和创新】
☐ 是否提出了新的理论或方法？
☐ 研究设计和方法是否严谨？
☐ 结论和建议是否有实用价值？

【快速评分系统】
根据以上标准，我给论文评分（满分10分）：
- 9-10分：必读，深入阅读
- 7-8分：应该读，了解观点
- 5-6分：可以读，补充知识
- <5分：可能不需要重点关注

请给我一个快速评分表的模板。
```

## 三、构建文献综述框架

### Prompt 3.1 - 帮我分类整理找到的论文

```
我已经找到了一些相关论文，现在需要给它们分类。

我的研究选题是：[选题]

请帮我设计一个论文分类体系。

【要求】
1. 根据我的研究主题，创建一个论文分类框架
   - 这个框架应该有3-5个大类
   - 每个大类下有2-3个小类
   - 这个分类应该能帮助我构建文献综述

2. 给出每个类别的定义和特点
   - 这类论文研究什么
   - 为什么对我的研究很重要
   - 应该重点关注这类中的哪些论文

3. 给我一个论文整理表格的模板：
   - 论文标题
   - 作者
   - 出版年份
   - 期刊/会议
   - 主要观点或贡献
   - 分类类别
   - 相关性评分（1-5分）
   - 是否已读
   - 关键引用

【示例】
比如你的分类可能是：
- 类别A：理论基础类
  └─ A1: 经典理论论文
  └─ A2: 理论延展论文
- 类别B：实证研究类
  └─ B1: 定量研究
  └─ B2: 定性研究
- ...等等

请根据我的选题设计具体的分类体系。
```

### Prompt 3.2 - 生成文献综述的框架和大纲

```
我已经读了30多篇相关论文，现在需要写文献综述。

我的研究选题是：[选题]

请帮我设计一个文献综述的框架：

【第一部分：综述框架设计】
1. 根据我的选题，应该怎样组织文献综述？
   - 按时间顺序？
   - 按理论学派分类？
   - 按研究方法分类？
   - 按其他维度？

2. 推荐的组织逻辑是什么？

【第二部分：详细大纲】
根据推荐的逻辑，给我一个详细的文献综述大纲：

例如：
第一章：理论基础
  1.1 经典理论
    - [理论1]：主要内容、代表学者、应用范围
    - [理论2]：...
  1.2 理论发展与演变
    - [如何从理论1发展到理论2]
  1.3 理论局限与评价

第二章：实证研究现状
  2.1 国外研究现状
    - [研究方向1]：主要发现、代表论文
    - [研究方向2]：...
  2.2 国内研究现状
    - ...
  2.3 中外研究对比

第三章：研究空白与问题
  3.1 理论层面的空白
  3.2 实证层面的空白
  3.3 我的研究如何填补这些空白

【第三部分：每一部分应该重点讨论什么】
对大纲的每个部分，标注：
- 应该讨论什么内容
- 用哪些论文作为支撑
- 应该突出什么观点

这会帮我快速撰写文献综述。
```

## 五、高效查找论文的实战指南

### Prompt 5.1 - 制定个人的论文查找计划

```
我要系统地查找和整理我研究领域的论文。

我的选题是：[选题]
我预计投入的时间：[几周/几个月]
我的目标是：[找到100篇/50篇/30篇核心论文]

请帮我制定一个详细的、可执行的论文查找计划：

【第一周：准备阶段】
任务1：确定搜索关键词
  - 进行[Prompt 2.1]来生成搜索策略
  - 时间：1-2小时
  - 输出：关键词清单

任务2：确定目标期刊和会议
  - 进行[Prompt 1.1]
  - 时间：1小时
  - 输出：期刊和会议清单

任务3：建立论文管理系统
  - 选择一个工具（如Zotero、Mendeley、NotionAI等）
  - 时间：30分钟
  - 输出：配置好的文献管理工具

【第二周：大规模搜索阶段】
任务1：在Google Scholar中搜索
  - 用关键词进行10-20次搜索
  - 收集100-200篇候选论文
  - 时间：4-6小时

任务2：在Web of Science/Scopus中搜索
  - 验证和补充高质量论文
  - 时间：2-3小时

任务3：在专业数据库中搜索
  - 如[学科专业数据库]
  - 时间：2-3小时

【第三周及以后：精筛和深入阶段】
任务1：初步筛选
  - 根据题目和摘要筛选，从100-200篇精选到50-80篇
  - 时间：3-4小时
  - 用[Prompt 2.2]帮你筛选

任务2：分类和组织
  - 把50-80篇论文分类
  - 用[Prompt 3.1]生成分类体系
  - 时间：2-3小时

任务3：精读重要论文
  - 精读前20-30篇最重要的论文
  - 用[Prompt 4.1]快速提炼每篇论文
  - 时间：10-15小时

任务4：浅读参考论文
  - 快速浏览其他论文的摘要
  - 时间：5-8小时

【第四周及以后：整合阶段】
任务1：构建文献综述框架
  - 用[Prompt 3.2]生成大纲
  - 时间：2-3小时

任务2：整理共同观点
  - 用[Prompt 4.2]对比多篇论文
  - 时间：3-5小时

任务3：撰写文献综述
  - 基于上述工作进行综述写作
  - 时间：5-10小时

【总体建议】
- 不要试图一次性找到所有论文，要分阶段
- 边搜边读，及时调整搜索方向
- 建立好论文管理工具，节省时间
- 定期总结，形成思路
```

## 八、快速参考：完整的论文查找流程

### Prompt 8.1 - 一个完整的论文查找流程（从头到尾）

```
从现在开始，我想一步步用你的帮助来找论文和写文献综述。

我的选题是：[选题]

请按照以下步骤，每一步都给我具体的指导和输出：

【步骤1】确定搜索策略（30分钟）
→ 用Prompt 1.1生成期刊和会议清单
→ 用Prompt 2.1生成搜索关键词和搜索式

【步骤2】大规模搜索（4-6小时）
→ 在Google Scholar中搜索（用步骤1的关键词）
→ 在Web of Science中补充搜索
→ 收集100-200篇候选论文
→ 输出：论文清单（标题、作者、期刊、年份）

【步骤3】初步筛选（2-3小时）
→ 根据题目和摘要筛选，从100-200篇精选到50-80篇
→ 用Prompt 2.2的标准评估每篇论文
→ 输出：精选论文清单（50-80篇，按重要性排序）

【步骤4】分类整理（2小时）
→ 用Prompt 3.1生成分类体系
→ 把50-80篇论文分类
→ 输出：按类别组织的论文清单

【步骤5】精读关键论文（5-8小时）
→ 精读前20-30篇最重要的论文
→ 对每篇用Prompt 4.1快速提炼
→ 输出：每篇论文的"论文速读卡"

【步骤6】对比分析（2-3小时）
→ 选择5-10篇最相关的论文
→ 用Prompt 4.2对比它们的观点
→ 输出：共同观点和分歧分析

【步骤7】生成文献综述框架（1-2小时）
→ 用Prompt 3.2生成文献综述大纲
→ 输出：详细的文献综述框架

【步骤8】撰写文献综述
→ 基于框架进行撰写
→ 引用之前整理的论文信息

【总耗时】
18-28小时 vs 传统方式的40-60小时
时间节省：50-70%

这个流程适合所有学科和所有阶段的学生。
```

## 十、常见问题和解决方案

### Q1: 我找到的论文太多了，怎么办？

**A:** 分阶段筛选
```
第1轮筛选：看题目 → 从500篇→100篇
第2轮筛选：看摘要 → 从100篇→50篇
第3轮筛选：看期刊等级和引用次数 → 从50篇→20篇
第4轮筛选：精读，判断相关性 → 从20篇→10篇
```

用Prompt 2.2加速这个过程。

---

### Q2: 找到的论文都是英文的，怎么办？

**A:**
- 优先读有中文摘要的（如果是中文期刊翻译版）
- 用翻译工具（Google翻译、DeepL等）快速翻译摘要
- 对最重要的5-10篇论文，值得花时间精读英文原文
- 用Prompt 4.1快速理解英文论文的核心

---

### Q3: 有些论文我找不到全文怎么办？

**A:** 多种渠道
- 在Google Scholar上看是否有免费版本
- 到ResearchGate上搜索这个作者，可能能下载
- 问你的学校图书馆（看是否订阅了这个期刊）
- 给作者发邮件请求论文副本（很多作者愿意分享）

---

### Q4: 我应该精读还是泛读？

**A:** 分层次
- **精读**（5-10篇）：对你选题最相关的、最重要的
  → 花2-3小时深入理解
- **重点阅读**（10-20篇）：比较相关的
  → 花30-45分钟，重点看方法和结论
- **泛读**（20-50篇）：参考性的
  → 花15-20分钟，看摘要和关键发现

---

### Q5: 怎样避免遗漏重要论文？

**A:**
- 用多个关键词搜索（最少5-10个）
- 用不同的数据库搜索（Google Scholar + Web of Science + 专业库）
- 查看你找到的重要论文的"被引用者"和"相关论文"
- 看前人的文献综述，看他们引用了哪些论文

---

## 总结：快速查找论文的黄金路径

```
【5分钟】确定搜索关键词
  ↓
【2小时】在Google Scholar中大规模搜索
  ↓
【1小时】初步筛选（300篇→50篇）
  ↓
【1小时】分类整理（按类别组织）
  ↓
【5小时】精读20篇最重要的论文
  ↓
【2小时】生成文献综述框架
  ↓
【5小时】撰写文献综述

总计：16小时
```

这样的流程能帮你在最短时间内找到最有价值的论文！
