# 论文选题AI提示词库

## 📋 使用说明

这是一个完整的Prompt库，可以直接用于ChatGPT、Claude等大模型。

**如何使用：**
1. 复制相关的Prompt
2. 根据你的具体情况填写[  ]中的内容
3. 粘贴到AI工具中运行
4. 对结果进行筛选和调整

---

## 一、选题生成类Prompt

### Prompt 1.1 - 通用选题生成（推荐）

**使用场景：** 初期广泛生成选题候选

```
我是一名[年级]学生，[专业]专业

【基本信息】
- 导师的研究方向是：[导师方向]
- 我的个人研究兴趣是：[你的兴趣]
- 我有的独特资源是：[你的资源/背景]
- 我想避免的研究方向是：[想避免的]

【要求】
请为我生成100个可能的论文选题。

要求条件：
1. 选题要具有学术价值和创新性
2. 选题要可行（研究生2-3年能完成）
3. 尽可能覆盖不同的研究角度（理论、实证、案例、综述等）
4. 每个选题都应该有明确的研究对象和研究问题
5. 优先考虑我的研究兴趣和资源优势

【输出格式】
请用表格形式列出所有选题，包含以下列：
- 序号
- 选题名称（简洁明确）
- 研究类型（理论/实证/案例/技术/综述）
- 主要研究问题
- 创新点/研究空白
- 难度评估（容易/中等/困难）
- 所需资源评估（少/中/多）

【后续整理】
在列出所有100个选题后，请：
1. 总结按难度划分的选题分类
2. 指出其中最有创新性的10个选题
3. 指出其中最容易实现的10个选题
4. 指出最适合我个人背景的10个选题
```

---

### Prompt 1.2 - 针对特定方向的深化选题

**使用场景：** 想进一步扩展某个特定方向

```
我对以下研究方向比较感兴趣：[方向描述]

请为这个方向生成50个更加具体和细化的选题。

具体要求：
1. 从以下多个角度考虑选题：
   - 研究对象角度（不同的群体、行业、组织）
   - 研究问题角度（不同的现象、问题、机制）
   - 时间角度（纵向变化、阶段比较）
   - 地域角度（不同区域、国家的比较）
   - 方法角度（定性、定量、混合、案例等）

2. 避免选题过于宽泛或过于具体
   - 太宽泛：难以在规定时间完成
   - 太具体：可能缺乏学术价值

3. 考虑到选题的新颖性和可行性的平衡

【输出要求】
- 分类呈现这50个选题
- 每个选题标注建议的研究方法
- 标注每个选题的理论基础或理论意义
- 标注每个选题的难度和周期估计
```

### Prompt 1.3 - 行业特定选题生成

**使用场景：** 针对特定行业或领域的选题

```
我想要做一个关于[具体行业/领域]的研究

行业背景信息：
- 行业名称：[行业]
- 行业特点：[主要特点]
- 行业面临的主要问题：[问题列表]
- 最近的行业发展趋势：[趋势]

请生成80个与这个行业相关的论文选题，要求：
1. 选题与该行业的实际问题相关
2. 既有理论意义，也有实践意义
3. 考虑不同的研究层次（微观个体、中观组织、宏观行业）
4. 考虑新技术对该行业的影响

输出格式：
- 按行业问题领域分类（如：效率、创新、人力资源等）
- 每个领域下列出相关的选题
- 为每个选题标注其实践应用价值
```

## 二、选题优化类Prompt

### Prompt 2.1 - 选题清晰化

**使用场景：** 当选题太宽泛或太模糊时

```
我的初步选题是：[你的选题]

我觉得这个选题有点：
☐ 太宽泛，范围不够清晰
☐ 太模糊，研究问题不具体
☐ 太复杂，涉及太多变量
☐ 其他问题：[描述]

请帮我改进这个选题：
1. 确定具体的研究对象（人群、组织、行业等）
2. 明确具体的研究问题（不要用"研究……的影响"这样的模糊表述）
3. 控制研究范围（不要涉及太多变量）
4. 确保有研究空白（这个角度是否前人已经研究过）

请给我3-5个改进版本的选题，并解释每个版本的改进点。
```

---

### Prompt 2.2 - 创新点挖掘

**使用场景：** 让选题更有创意和新意

```
我现在考虑的选题是：[你的选题]

我觉得这个选题的创新点还不够明显。

请帮我：
1. 分析这个选题与已有研究的关系
   - 哪些方面已经被前人研究过？
   - 哪些方面还是研究空白？

2. 挖掘可能的创新角度，包括：
   - 新的研究对象（换个群体、行业、地域）
   - 新的研究视角（换个理论框架或学科视角）
   - 新的研究问题（从不同层面提问题）
   - 新的研究方法（用新的研究方法）
   - 新的时间维度（纵向变化而非横截面）

3. 提出3-5个具有更强创新性的选题变体

4. 对每个变体说明其创新点的强度（弱/中等/强）
```

---

### Prompt 2.3 - 可行性评估

**使用场景：** 评估选题是否真的可行

```
我想评估以下选题的可行性：[你的选题]

我的研究条件：
- 研究周期：[几年]
- 可用的研究方法技能：[你掌握的方法]
- 可获取的资源：[你能获取的数据、设备、人力等]
- 时间投入：[每周能投入的时间]

请帮我：
1. 分析这个选题所需的资源和条件
2. 列出主要的研究困难
3. 为每个困难提出可能的解决方案
4. 评估该选题的可行性（1-10分）
5. 如果可行性不足，提出如何改进选题来提高可行性
```

## 三、选题分析类Prompt

### Prompt 3.1 - 深度学术分析

**使用场景：** 深入理解选题的学术基础

```
我想深度分析以下论文选题：
【选题：[你的选题]】

请从以下方面帮我进行深度分析：

【第一部分：学术背景】
1. 这个选题的理论基础是什么？
   - 涉及哪些经典理论？
   - 这些理论如何解释这个问题？

2. 这个问题为什么重要？
   - 学术意义在哪里？
   - 实践意义在哪里？

【第二部分：研究现状】
3. 已有哪些相关研究？
   - 请列出3-5个最相关的研究方向
   - 简述每个方向的主要发现

4. 已有研究的观点有什么分歧？
   - 不同的理论流派怎样看待这个问题？
   - 实证研究的发现是否一致？

5. 已有研究的主要空白在哪里？
   - 理论层面还有什么没解释清楚？
   - 实证证据还不足的地方？
   - 缺少什么样的研究对象或背景？

【第三部分：创新点分析】
6. 这个选题相比现有研究的创新在哪里？
   - 是新视角、新对象还是新方法？
   - 这个创新有多明显（弱/中等/强）？

7. 这个创新点容易被导师和学术界认可吗？
   - 为什么值得研究？
   - 是否有可能过度宣传了创新性？

【第四部分：理论框架建议】
8. 用什么理论框架来研究这个问题？
   - 主要的理论基础是什么？
   - 需要什么样的概念框架？

9. 核心的研究变量有哪些？
   - 自变量、因变量、中介变量、调节变量分别是什么？
   - 这些变量如何测量？

【第五部分：方法建议】
10. 适合用什么研究方法？
    - 定性、定量还是混合？
    - 为什么选择这个方法？

11. 应该怎样收集数据？
    - 样本应该选择谁？
    - 样本量需要多大？
    - 如何确保样本代表性？

【第六部分：可行性分析】
12. 这个研究可能面临什么具体困难？
    - 方法上的困难？
    - 数据获取的困难？
    - 时间上的困难？

13. 这些困难有什么解决方案吗？
    - 技术方案？
    - 时间安排？
    - 备选方案？

【第七部分：周期预估】
14. 完成这个研究大概需要多长时间？
    - 文献综述：？
    - 理论框架构建：？
    - 研究设计：？
    - 数据收集：？
    - 数据分析：？
    - 论文撰写：？
    - 修改完善：？
    - 总计：？

15. 在2-3年内完成的概率有多大？

【第八部分：成果预期】
16. 完成这个研究会产生什么成果？
    - 学位论文是否会有竞争力？
    - 能否发表学术论文？发表什么级别？
    - 是否有实践应用价值？

【总体评价】
17. 综合上述分析，这个选题的综合评分是多少？
    - 学术价值：？/10
    - 创新性：？/10
    - 可行性：？/10
    - 总体推荐度：？/10
```

### Prompt 3.2 - 对比分析多个选题

**使用场景：** 在几个选题之间难以选择时

```
我现在在这几个选题之间犹豫：
1. [选题1]
2. [选题2]
3. [选题3]

请帮我从多个维度对比这些选题：

【对比维度】
1. 创新性对比
   - 哪个选题最创新？为什么？
   - 创新点的强度排序？

2. 可行性对比
   - 哪个选题最容易完成？
   - 各自的主要困难是什么？

3. 学术价值对比
   - 哪个选题最有学术价值？
   - 对该领域的贡献大小排序？

4. 兴趣度对比
   - 根据我的描述，我对哪个最感兴趣？
   - 这个兴趣能否持续3年？

5. 资源需求对比
   - 哪个选题所需资源最少？
   - 各自的资源短板是什么？

【综合评估】
6. 如果要用5维度综合评分（创新20% + 可行性25% + 学术价值30% + 兴趣15% + 资源10%），各个选题的综合得分是多少？

7. 最后的推荐是什么？为什么？

8. 如果非要在这三个之间选一个，我应该选哪个？理由是什么？
```

## 四、选题论证类Prompt

### Prompt 4.1 - 生成选题论证方案

**使用场景：** 准备和导师讨论时

```
我的论文选题是：[选题]

请帮我生成一个完整的【选题论证方案】，包括以下内容：

【一、研究背景与意义】
1. 理论意义
   - 这个研究在理论上填补了什么空白？
   - 有什么理论创新或理论贡献？

2. 现实意义
   - 现实中存在什么问题或现象？
   - 这个研究如何帮助解决实践问题？

【二、研究问题与创新点】
3. 核心研究问题
   - 主问题是什么？用最简洁的语言表述
   - 有什么子问题？

4. 创新点
   - 与前人研究相比，创新在哪里？
   - 创新点有多明显？

【三、研究框架与方法】
5. 理论框架
   - 用什么理论来解释这个问题？
   - 核心变量有哪些？
   - 变量之间的关系如何？

6. 研究方法
   - 用什么研究方法？（定性/定量/混合）
   - 为什么选择这个方法？
   - 主要分析方法是什么？

【四、研究可行性】
7. 数据和样本
   - 研究需要什么样的数据？
   - 样本选择的逻辑是什么？
   - 样本量是多少？
   - 数据如何获取？有什么困难吗？

8. 研究周期
   - 各个阶段的时间安排？
   - 总共需要多长时间？
   - 在2-3年内能完成吗？

9. 资源需求
   - 需要什么资源？
   - 学校/导师能提供吗？
   - 有什么资源短板吗？

【五、预期成果】
10. 主要成果
    - 会产生什么研究成果？
    - 预期的论文质量？
    - 能否发表高质量论文？

11. 学术贡献
    - 对理论的贡献是什么？
    - 对实践的指导意义是什么？

最后，整理成一个2-3页的论文，用于和导师讨论。
```

### Prompt 4.2 - 生成导师讨论稿

**使用场景：** 准备和导师的正式讨论

```
我要和导师讨论我的论文选题。

我的选题是：[选题]
我的初步想法是：[简述你的思路]

请帮我准备一份【导师讨论稿】，包括：

【一、开场（2分钟说清楚）】
1. 我的选题是什么？（一句话概括）
2. 为什么选这个题目？（个人兴趣+学术价值）
3. 我的初步思路是什么？（研究方向、研究对象、研究方法的简述）

【二、深入讨论的准备（预留导师提意见的时间）】
4. 可能的讨论点：
   - 选题是否过大或过小？
   - 研究方法是否合理？
   - 数据获取是否可行？
   - 创新性是否足够？

【三、备选方案（以防导师不满意）】
5. 如果导师建议修改选题，我可以考虑的备选方向是什么？

【四、需要导师帮助的地方】
6. 我需要导师在以下方面的帮助：
   - 数据获取（企业联系、问卷发放等）
   - 方法指导（某种研究方法的使用）
   - 资源协助（设备、资金等）

最后，生成一个可以直接用的讨论稿（800-1000字）。
```

## 六、选题修改和优化提示

### 当导师反馈"选题太宽泛"

```
导师反馈我的选题太宽泛。我的原始选题是：[选题]

请帮我缩小选题范围，方式包括：
1. 缩小研究对象（如：从"企业"缩小到"互联网企业"或"初创企业"）
2. 缩小研究问题（如：从"影响"缩小到"特定方面的影响"）
3. 缩小时间范围（如：从"长期变化"改为"特定时期"）
4. 缩小地理范围（如：从"全国"改为"特定地区"）

请给我3-5个更具体的选题修改版本，并解释每个版本缩小的方向。
```

---

### 当导师反馈"创新点不足"

```
导师说我的选题创新点不够明显。

原始选题：[选题]
导师的具体反馈是：[导师的意见]

请帮我增强选题的创新性，可能的方向包括：
1. 引入新的理论角度
2. 研究新的群体或对象
3. 采用新的研究方法
4. 提出新的研究问题
5. 融合新的研究元素

请给我3-5个改进版本，并对每个版本标注增强的创新性亮点。
```

---

### 当导师反馈"数据获取困难"

```
导师指出我的选题需要的数据难以获取。

## 八、论文选题最佳实践清单

**在最终确定选题前，检查以下清单：**

```
☐ 选题是否足够清晰具体？
  - 能用一句话明确说出研究对象和研究问题吗？

☐ 选题是否有研究空白？
  - 这个角度是否前人已经充分研究过？
  - 你的选题填补了什么空白？

☐ 选题是否有创新点？
  - 创新是新角度、新对象还是新方法？
  - 创新程度有多明显？

☐ 选题是否有学术价值？
  - 对理论有什么贡献？
  - 对实践有什么启示？

  - 方法是否掌握或容易学？

## 使用建议

1. **第一次用**：建议使用Prompt 1.1 (通用选题生成)
2. **选题太多**：使用Prompt 2.1 (选题清晰化)和Prompt 2.2 (创新点挖掘)
3. **在几个选题间摇摆**：使用Prompt 3.2 (对比分析)
4. **准备和导师讨论**：使用Prompt 4.2 (导师讨论稿)
5. **需要修改选题**：根据导师反馈，选用Prompt 5的相应部分

**一个完整的选题周期可能的Prompt使用顺序：**
1. Prompt 1.1 → 生成100个选题
2. Prompt 2.1或2.2 → 优化和调整
3. Prompt 3.2 → 在最好的几个中对比
4. Prompt 3.1 → 深度分析最有希望的选题
5. Prompt 4.2 → 准备和导师讨论
6. 根据导师反馈，可能重复上述步骤
