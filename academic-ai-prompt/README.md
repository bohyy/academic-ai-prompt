## 📖 项目介绍

这是一套为**研究生、学术研究者**设计的完整AI Prompt合集，涵盖：

- ✅ **论文选题** - 用AI系统地生成和评估论文选题
- ✅ **期刊期会议查找** - 快速找到你研究领域的Top期刊和会议
- ✅ **论文检索** - AI直接帮你列举相关论文
- ✅ **文献综述** - 系统整理和对比论文
- ✅ **研究方法** - AI辅助研究设计
- ✅ **论文写作** - 从构思到成稿

**核心优势：**
- 🚀 **节省时间** - 使用这套方法可节省50-70%的时间
- 📚 **科学方法** - 基于系统的思维框架
- 💡 **即插即用** - 所有Prompt都可直接复制使用
- 🎯 **全流程覆盖** - 从选题到出版的全过程

## 🚀 快速开始

### 方案1️⃣：我要选论文题目

**第1步** → 打开 `01_论文选题系列/论文选题AI提示词库.md`

**第2步** → 复制 `Prompt 1.1`（生成100个选题）

**第3步** → 粘贴到 ChatGPT/Claude，填入你的信息

**第4步** → 用 `论文选题5维度评估表.xlsx` 评分筛选

**时间** → 3-5天完成

---

### 方案2️⃣：选题已定，我要找论文

**第1步** → 打开 `02_论文查找系列/AI直接找论文的Prompt.md`

**第2步** → 选择方案八（最快版）或方案一（标准版）

**第3步** → 把你的选题填入Prompt，发给AI

**第4步** → AI直接给你30-50篇论文列表

**时间** → 5-10分钟获得论文清单

---

### 方案3️⃣：我要系统地做文献综述

**第1步** → 先用方案2️⃣找到50-100篇论文

**第2步** → 打开 `02_论文查找系列/论文查找和文献综述AI提示词库.md`

**第3步** → 用 `Prompt 3.1` 和 `Prompt 3.2` 分类和生成框架

**第4步** → 撰写文献综述

**时间** → 15-20小时完成

## 📊 核心方法论

### 🎯 五维度论文选题评估

```
创新性(20%) + 可行性(25%) + 学术价值(30%) + 兴趣度(15%) + 资源性(10%)
= 加权总分

8.0+  → 强烈推荐
7.0-7.9 → 很好
6.0-6.9 → 可以
<6.0   → 不推荐
```

### 🔍 论文查找黄金路径

```
5分钟  → 确定搜索关键词
   ↓
2小时  → Google Scholar大规模搜索
   ↓
1小时  → 初步筛选（300篇→50篇）
   ↓
5小时  → 精读20篇关键论文
   ↓
2小时  → 生成文献综述框架
   ↓
5小时  → 撰写文献综述

【总计】16小时 vs 传统40-60小时
【节省】50-70%时间
```

## 💡 典型使用场景

### 场景1：我是研究生，需要选论文题目

**时间投入：** 3-5天
**步骤：**
1. 使用 `论文选题AI提示词库.md` 中的 Prompt 1.1 生成100个选题
2. 用 `论文选题5维度评估表.xlsx` 评分和筛选

## 💬 使用建议

### ✅ 怎样最有效地使用

1. **按顺序来** - 先选题，再找论文，再写综述
2. **结合人工判断** - AI的输出是参考，最终决定权在你
3. **多尝试** - 不同的Prompt可能给出不同的结果，试试多个
4. **持续优化** - 根据效果调整Prompt中的参数

### ⚠️ 注意事项

- ⚠️ AI的输出可能有时不完全准确，需要在Google Scholar中验证
- ⚠️ 论文列表中的某些论文可能已过时，需要确认发表年份
- ⚠️ 不同学科的情况不同，某些Prompt可能需要根据学科特点调整

1. 复制一个Prompt到ChatGPT/Claude

## 📞 常见问题

### Q: 这些Prompt是否适用于所有学科？
**A:** 基本上适用，但不同学科可能需要微调。我们提供了多个版本，包括"专业特定版"。

### Q: AI生成的论文清单是否准确？
**A:** AI的输出是基于训练数据的最佳判断，建议在Google Scholar中验证。我们的方法是：AI提供初步清单 → 人工验证 → 获得最终论文库。

### Q: 可以直接用这些Prompt发布到社交媒体吗？
**A:** 可以，但建议标注来源。如果公开分享，请注明是来自本项目。

### Q: 如果我的导师不同意用AI怎么办？
**A:** AI只是辅助工具，所有最终的学术判断和决定权都在你和导师手里。我们的方法论和思维框架同样有价值。
